Banking-system-cpp
A console-based banking system in C++ using OOP (classes, encapsulation, file handling) supports account creation, deposits, withdrawals, balance checks, and transaction history.
Features:
1.Create bank accounts with PIN protection
2.Deposit & withdraw money with balance validation
3.Check account balance securely
4.View full transaction history with timestamps
5.Delete accounts
6.All data saved to file and reloaded automatically on next run
Concepts Used:
      Classes & Objects Encapsulation Constructors File Handling (fstream) Vectors Modular Design
